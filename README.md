#  R.O.A.D.S

## Responsive Odisha Adaptive Driving Systems

###  Smart India Hackathon 2025 Prototype

---

##  Overview

**R.O.A.D.S** is an AI-powered smart traffic management system designed to reduce urban congestion using real-time computer vision and intelligent analytics.

It leverages advanced **YOLO-based object detection** and scalable backend systems to monitor, analyze, and optimize traffic flow across intersections.

##  Key Features

### Real-Time Monitoring

* Live vehicle tracking across multiple intersections
* Multi-camera feed integration

### Smart Analytics

* Vehicle count detection
* Hard braking detection
* Tailgating alerts
* Average speed estimation
* Platoon density analysis

### Interactive Dashboard

* Clean UI for traffic insights
* Graphs & real-time updates
* Decision-support visualization

### Scalable Architecture

* Designed for city-wide deployment
* Handles multiple video streams efficiently

---

##  How It Works

1.  Video feeds captured from traffic cameras
2.  YOLO model detects and tracks vehicles
3.  FastAPI processes real-time data
4.  Dashboard visualizes analytics
5.  Authorities take intelligent decisions
